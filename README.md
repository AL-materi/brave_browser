# Brave Browser - Flutter Project

## المميزات:

1. **Splash Screen** - شاشة بداية لمدة 2 ثانية
2. **تسجيل الدخول** - اسم مستخدم وكلمة مرور
3. **البحث** - مع نقل النص لصفحة السجل
4. **السجل** - عرض عمليات البحث في كروت
5. **تغيير صورة البروفايل** - باستخدام image_picker
6. **الإعدادات** - مع sliders للتحكم

## Packages المستخدمة:

- `image_picker: ^1.0.4` - لاختيار الصور
- `Slider المدمج في Flutter` - للتحكم في الإعدادات

## كيفية التشغيل:

```bash
flutter clean
flutter pub get
flutter run
```
